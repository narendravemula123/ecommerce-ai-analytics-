# 🚀 E-Commerce AI Analytics System

An end-to-end data analytics pipeline for e-commerce sales data with KPI generation, automated reporting, and AI-powered product description generation.

## 🔧 Features

* 📊 Revenue analysis & KPI generation
* 🏆 Top product & category insights
* 📈 Category contribution analysis
* 🤖 AI-generated product descriptions using OpenAI API
* 📁 Automated Excel report generation

## 🛠 Tech Stack

* Python (Pandas)
* OpenAI API
* Excel Automation (openpyxl)
* Data Analytics

## 📂 Project Structure

* `Data/` → Input dataset
* `Scripts/` → Core logic
* `Output/` → Generated reports

## ▶️ How to Run

```bash
python Scripts/main.py
```

## 🔐 Note

API keys are handled securely using environment variables.
